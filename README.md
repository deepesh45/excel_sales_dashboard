# Sales Dashboard using Microsoft Excel

## Project Overview
This project is an interactive Sales Dashboard developed using Microsoft Excel to analyze sales performance across different regions, categories, and customer segments.

The dashboard helps in understanding:
- Total Sales
- Quantity Sold
- Regional Performance
- Product Category Analysis
- Discount Impact

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting

## Dataset Information
The dataset contains:
- Order ID
- Order Date
- Customer Details
- Product Category
- Sales
- Quantity
- Discount
- Region & State Information

## Key Insights
- Technology category generated high sales.
- Central and West regions showed strong performance.
- Consumer segment contributed major quantity sales.
- Discounts impacted sales trends in several categories.

## Dashboard Features
- Interactive filtering using slicers
- Sales comparison by category
- Quantity analysis by customer segment
- Region-wise sales tracking
- Dynamic charts and KPI visualization
